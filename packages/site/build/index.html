工作原理

01

安装 CLI 工具

通过 npm 或 pnpm 全局安装 AIUsage。轻量的命令行工具，不常驻后台。

02

解析日志文件

运行 aiusage parse，自动发现并解析 AI 工具生成的本地日志文件，写入 SQLite 数据库。

03

查看仪表盘

运行 aiusage serve 启动本地 Web 仪表盘。浏览器打开 localhost:3847，所有数据一目了然。

为什么选择本地优先？

●
数据不出本机

所有日志解析和数据存储都在本地完成。SQLite 数据库文件就在你的硬盘上。

●
零依赖服务

不需要注册账号，不需要 API Key，不需要云服务。装好就能用。

●
可选的多设备同步

通过你自己的 GitHub 仓库或 S3 存储同步，数据始终在你的控制下。
